A collection of hands-on SAP Cloud Integration (SAP CPI) projects built while learning SAP Integration Suite and enterprise integration patterns.

## Technologies & Concepts

* SAP Integration Suite (SAP CPI)
* Integration Flows (iFlows)
* Content Modifiers
* Request-Reply Pattern
* Routing & Conditional Logic
* XML Processing & XPath
* JSON ↔ XML Transformations
* Data Store Operations
* Groovy Scripting
* Message Mapping
* Monitoring & Logging
* Enterprise Integration Patterns

## Projects

* HelloWorld iFlow
* XML Processing & XPath Expressions

Each project is maintained in its own folder and includes:

Build practical SAP CPI integration skills through real-world scenarios and create a portfolio of integration projects demonstrating enterprise integration concepts.
